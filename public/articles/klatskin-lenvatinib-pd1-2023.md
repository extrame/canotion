# 胆道介入联合仑伐替尼+PD-1抑制剂治疗肝门部胆管癌合并恶性梗阻性黄疸

**来源：** Journal of Clinical Oncology (ASCO 2023)
**原文链接：** [ASCO Publications](https://ascopubs.org/doi/10.1200/JCO.2023.41.16_suppl.e14620)
**DOI：** 10.1200/JCO.2023.41.16_suppl.e14620

---

## 一、研究背景

肝门部胆管癌（Klatskin肿瘤）是一种罕见且高度侵袭性的恶性肿瘤，发生于左右肝管汇合处或附近。吉西他滨+顺铂已被确立为有效的一线治疗方案。

然而，该恶性肿瘤常合并恶性胆道梗阻，可能逐渐损害患者肝功能并限制化疗的应用。此外，累积的临床证据表明，酪氨酸激酶抑制剂（TKI）联合免疫治疗是一种有前景的替代方案。

本研究旨在评估胆道介入联合仑伐替尼+PD-1抑制剂（替雷利珠单抗/信迪利单抗）治疗肝门部胆管癌的临床价值。

## 三、研究方法

### 研究设计

| 项目 | 内容 |
| --- | --- |
| **研究类型** | 回顾性临床研究 |
| **研究时间** | 2020年1月至2022年12月 |
| **入组患者** | 21例Klatskin肿瘤合并恶性肝门部胆道梗阻患者 |
| **干预措施** | 金属支架+Iodine-125粒子条植入，序贯仑伐替尼+PD-1抑制剂 |

### 胆道介入方案

所有入组患者接受金属支架+Iodine-125粒子条（ISS）植入进行胆道减压。

- **Iodine-125粒子数量**：根据术前计算机治疗计划系统（TPS）计划确定
- **成功引流定义**：术后4周内血清总胆红素水平下降至术前值的75%以下

### 系统治疗方案

| 药物 | 类别 |
| --- | --- |
| **仑伐替尼** | 多靶点酪氨酸激酶抑制剂（TKI） |
| **替雷利珠单抗** | PD-1抑制剂 |
| **信迪利单抗** | PD-1抑制剂 |

## 四、研究结果

### 引流效果

| 指标 | 结果 |
| --- | --- |
| **成功引流率** | 71.4%（15/21） |
| **总胆红素改善** | 术后1个月显著降低（P<0.001） |
| **直接胆红素改善** | 术后1个月显著降低（P<0.001） |

### 并发症

| 并发症类型 | 发生率 | 具体情况 |
| --- | --- | --- |
| **总体轻微并发症** | 23.8%（5/21） | - |
| 胆管炎 | 4.8%（1/21） | 抗感染治疗后好转 |
| 腹痛 | 19%（4/21） | 对症处理后缓解 |
| **严重并发症** | 0% | 无手术相关死亡或严重不良事件 |

### 生存分析

| 分组 | 中位总生存期（mOS） | P值 |
| --- | --- | --- |
| **全部患者** | 6.1个月 | - |
| 成功引流组（n=15） | 7.4个月 | <0.001 |
| 引流失败组（n=6） | 3.5个月 | - |
| 引流后序贯治疗组（n=9） | 7.4个月 | 0.028 |
| 仅姑息治疗组（n=6） | 5.2个月 | - |

## 五、研究结论

胆道介入+Iodine-125粒子条植入联合后续仑伐替尼+PD-1抑制剂治疗对于合并梗阻性黄疸的胆管癌患者是**安全可行**的。

### 关键发现

1. **有效引流改善预后**：成功引流患者的中位生存期显著延长（7.4 vs 3.5个月）
2. **序贯治疗获益**：引流成功后接受仑伐替尼+PD-1抑制剂治疗的患者生存期进一步延长
3. **安全性良好**：无严重手术相关并发症

## 六、临床意义

### 对于恶性梗阻性黄疸患者

| 要点 | 说明 |
| --- | --- |
| **胆道引流优先** | 对于合并黄疸的晚期胆管癌患者，应优先考虑胆道引流恢复肝功能 |
| **Iodine-125粒子条** | 金属支架联合放射性粒子条可提供持续局部放疗 |
| **序贯系统治疗** | 引流成功后，肝功能恢复可考虑仑伐替尼+PD-1抑制剂联合治疗 |

### 治疗流程建议

```
Klatskin肿瘤 + 恶性梗阻性黄疸
        ↓
胆道介入（金属支架 + Iodine-125粒子条）
        ↓
    ┌───┴───┐
    ↓       ↓
引流成功   引流失败
    ↓       ↓
肝功能恢复  姑息治疗
    ↓
仑伐替尼 + PD-1抑制剂
    ↓
延长生存期
```

---

## 七、患者关注要点

1. **胆道引流的必要性**：对于黄疸患者，引流是后续治疗的基础
2. **引流成功标准**：胆红素下降至术前75%以下为成功
3. **Iodine-125粒子**：提供局部放疗，可能增强局部控制
4. **系统治疗时机**：需等待肝功能恢复后再开始仑伐替尼+PD-1治疗
5. **不良反应监测**：需关注胆管炎、腹痛等并发症
