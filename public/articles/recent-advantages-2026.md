## 一、pCCA的定义与分类

| 项目         | 内容                                       |
| ---------- | ---------------------------------------- |
| **解剖定义**   | pCCA（肝门部胆管癌）又称Klatskin肿瘤，起源于左右肝管汇合处的胆管上皮 |
| **分类归属**   | 属于肝外胆管癌的亚型，与远端胆管癌共同构成eCCA                |
| **流行病学问题** | 历史上常被错误分类为iCCA（肝内胆管癌），影响流行病学数据精确性        |
| **发病率**    | pCCA占全部胆管癌的50%以上，是临床研究的重点与难点             |

## 二、pCCA相关分子特征与靶向治疗

### 1. HER2扩增/过表达 — **pCCA的关键靶点**

| 特征       | 数据                                                                    |
| -------- | --------------------------------------------------------------------- |
| **发生率**  | HER2扩增/过表达在iCCA中罕见，但在**eCCA（含pCCA）和GBC中高达15-20%**                     |
| **治疗依据** | HER2靶向治疗策略在eCCA和GBC患者中具有明确合理性                                         |
| **获批药物** | Zanidatamab（双特异性HER2抗体）于2024年11月获FDA加速批准用于HER2阳性BTC                   |
| **临床数据** | HERIZON-BTC-01试验：Zanidatamab在HER2阳性（IHC 3+）患者中ORR达41.3%，中位缓解持续时间≥13个月 |

**对pCCA的意义：** 由于pCCA属于eCCA范畴，HER2靶向治疗对pCCA患者具有重要的临床价值，是当前最相关的分子靶点之一。

### 2. FGFR2融合/重排 — 主要见于iCCA

| 特征          | 数据                                   |
| ----------- | ------------------------------------ |
| **发生率**     | 约10-20%的iCCA病例携带FGFR2融合/重排           |
| **pCCA相关性** | **在pCCA中较少见**，主要见于肝内胆管癌              |
| **获批药物**    | Pemigatinib、Infigratinib、Futibatinib |

| **pCCA相关性** | **在pCCA中较少见**            |
| **获批药物**    | Ivosidenib（基于ClarIDHy试验） |

| **临床数据** | ROAR试验胆道癌队列：ORR达51%，中位PFS 9.0个月，中位OS 13.5个月 |

## 三、免疫治疗进展 — 适用于所有BTC亚型（含pCCA）

### 1. 一线治疗里程碑试验

| 试验名称            | 方案                       | 结果                            |
| --------------- | ------------------------ | ----------------------------- |
| **TOPAZ-1**     | Durvalumab + GC vs GC    | OS显著延长：12.9 vs 11.3个月；长期生存率提高 |
| **KEYNOTE-966** | Pembrolizumab + GC vs GC | OS显著延长：12.7 vs 10.9个月         |

**对pCCA的意义：** 免疫联合化疗已成为晚期BTC（包括pCCA）的一线标准治疗方案。

### 2. 二线治疗选择

| 方案                                  | 说明      |
| ----------------------------------- | ------- |
| Nanoliposomal irinotecan（纳米脂质体伊立替康） | 可行的二线选择 |
| 双免疫检查点抑制剂联合                         | 正在探索中   |

## 四、辅助治疗进展

| 试验         | 方案    | 结果                                        |
| ---------- | ----- | ----------------------------------------- |
| BILCAP     | 卡培他滨  | 中位OS：49.6 vs 36.1个月（对照组），但未达统计学显著性        |
| PRODIGE 12 | GEMOX | 中位OS：75.8 vs 50.8个月（HR=1.08，P=0.74），无显著差异 |

**现状：** 辅助治疗结果尚不明确，BTC的异质性（包括pCCA与其他亚型的差异）是主要挑战。

---

## 五、pCCA治疗要点总结

| 领域       | 关键进展                                               |
| -------- | -------------------------------------------------- |
| **分子靶点** | HER2扩增是pCCA最重要的可靶向改变（15-20%），FGFR2/IDH1突变在pCCA中较少见 |
| **一线治疗** | 免疫检查点抑制剂联合GC化疗是标准方案                                |
| **靶向治疗** | HER2阳性pCCA患者可考虑Zanidatamab等HER2靶向药物                |
| **辅助治疗** | 尚无明确标准，需个体化决策                                      |
| **未来方向** | 分子分型指导的精准治疗、生物标志物开发、多模式治疗优化                        |

---

## 六、临床实践建议

1. **分子检测**：所有pCCA患者应进行全面的分子谱检测，重点关注HER2状态
2. **一线治疗**：Durvalumab/Pembrolizumab + GC方案
3. **二线治疗**：根据分子改变选择靶向药物（HER2阳性→Zanidatamab）
4. **临床试验**：鼓励符合条件的pCCA患者参与临床试验
